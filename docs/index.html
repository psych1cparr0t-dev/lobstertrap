How it works

From install to live agent
in under 5 minutes.

LobsterTrap handles the plumbing — Docker, configs, secrets, integrations — so you can focus on what your agent actually does.

01
🔧

Install & initialize

One command to install LobsterTrap and set up OpenClaw locally. We check Docker, Python, and everything else is ready.

$ lobstertrap doctor
✓ Docker running
✓ Node.js v18+
✓ Python 3.9+
02
📦

Pick a template

Choose from Sales, CRM, or Support — or start from scratch. LobsterTrap generates your agent code, Dockerfile, and OpenClaw config.

$ lobstertrap new
? Template: Sales Agent
? Integrations: Gmail, Airtable
03
🚀

Deploy & monitor

One command builds the Docker image, starts the container, and waits for a healthy response. Open the dashboard and you're done.

$ lobstertrap deploy SalesBot
✓ Agent is healthy
$ lobstertrap dashboard
